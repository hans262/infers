backPropagation

  • backPropagation(hy: Matrix[], xs: Matrix, ys: Matrix): { dw: Matrix[]; dy: Matrix[] }
    • Defined in BPNet.ts:213

    单样本求导，对每个输出单元的求导，对每个权重的求导

    • J = 1 / 2 * (hy - y)^2
    • ∂J / ∂hy[last] = (1 / 2) * 2 * (hy - y) = (hy - y) * hy[激活函数求导]
    • ∂J / ∂w = dy.T * lastHy
    • ∂J / ∂hy[now] = (nextDy * nextW) * hy[激活函数求导]

    链式求导法则：

    • 反向的计算过程；
    • 分支节点相加；链路节点相乘法；
    • 如有激活函数需乘激活函数的导数；

    Parameters

    • hy: Matrix[]
    • xs: Matrix
    • ys: Matrix

    Returns { dw: Matrix[]; dy: Matrix[] }

    • dy: Matrix[]
