calcDerivative

  • calcDerivative(ys: Matrix, hy: Matrix[]): { dw: Matrix[]; dy: Matrix[] }
    • Defined in BPNet.ts:145

    求误差相对于每一个节点的导数

    • E = 1 / 2 (hy - y)^2
    • ∂E / ∂hy = (1 / 2) * 2 * (hy - y) = hy - y 最后一个节点的导数
    • ∂E / ∂w = 当前权重输入节点的值 * 输出节点的导数

    求导遵循链式法则：分支节点相加，链路节点相乘。 有激活函数，还需乘以激活函数的导数。

    Parameters

    • ys: Matrix
    • hy: Matrix[]

    Returns { dw: Matrix[]; dy: Matrix[] }

    • dy: Matrix[]
