calcDerivative

  • calcDerivative(hy: Matrix[], ys: Matrix, n: number): { dw: Matrix[]; dy: Matrix[] }
    • Defined in BPNet.ts:163

    The derivative of the valence function with respect to each neuron and the derivative of each weight are calculated. It's for a single sample.

    • J = 1 / 2 * (hy - y)^2
    • ∂J / ∂hy = (1 / 2) * 2 * (hy - y) = hy - y Derivation of the last node
    • ∂J / ∂w = Value of input node * derivative of output node

    Derivation follows the chain rule: branch nodes add, link nodes multiply. If there is an activation function, it needs to be multiplied by the derivative of the activation function.

    Parameters

    • hy: Matrix[]
    • ys: Matrix
    • n: number

    Returns { dw: Matrix[]; dy: Matrix[] }

crossCost

  • crossCost(hy: Matrix[], ys: Matrix): number
    • Defined in BPNet.ts:221

    Cross entropy cost function To simulate the last layer is the sigmoid activation function Multiple outputs average multiple loss values 输出值域必须是 {0, 1}

    • J = 1 / m * ∑m Cost
    • y = 1 ? Cost = - Math.log(H(X[i]))
    • y = 0 ? Cost = -Math.log(1 - H(X[i]))
