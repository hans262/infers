calcDerivative

  • calcDerivative(hy: Matrix[], ys: Matrix): { dw: Matrix[]; dy: Matrix[] }
    • Defined in BPNet.ts:161

    单样本求导，对每个输出单元的求导，对每个权重的求导

    • J = 1 / 2 * (hy - y)^2
    • ∂J / ∂hy = (1 / 2) * 2 * (hy - y) = hy - y 最后一层节点求导
    • ∂J / ∂w = 输入节点 * 输出节点导数 遵循链式求导法则：分支节点相加；链路节点相乘法；反向的计算过程 如果有激活函数，需乘激活函数的导数
